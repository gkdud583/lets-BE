# lets

[hola](https://holaworld.io/)를 클론 코딩한 스터디, 사이드 프로젝트 팀원을 구하는 웹 페이지

## 기술 스택
- JAVA 11
- Spring Boot 2.6.1
- Spring Secuity
- Spring Data JPA & QueryDSL 5.0.0
- Spring Data Redis
- H2 1.4.200
- JWT 0.9.0
- Cloudinary 1.0.2
- JUnit 5.8.1 & Mockito
